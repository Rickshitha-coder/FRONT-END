CSS Background Properties

Background Color

This box has a solid background color (green).

Background Image

This box has a background image. The image covers the entire box.

Background Repeat

This box has a repeated background image.

Background Position

This box centers the background image.

Background Gradient

This box has a linear gradient background that goes from pink to orange.

Background Attachment

This box has a fixed background image that stays in place while scrolling.

Combined Background

This box uses multiple background properties combined into one.
