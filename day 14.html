CSS Box Model

Content Only

This box has only the content area. No padding, no border, and no margin are applied. It appears tight with no extra spacing.

Content + Padding

This box includes padding. Padding adds space inside the box, between the content and the border (if there is one).

Content + Padding + Border

This box has a green border around the content and padding. Border visually outlines the box.

Content + Padding + Border + Margin

This final box includes margin as well. Margin is space outside the box, separating it from other elements.
